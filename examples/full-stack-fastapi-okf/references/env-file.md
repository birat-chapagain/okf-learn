# Environment File

## Relevant Facts

- `.env` includes domain, frontend host, environment, project name, stack name, CORS origins, backend secrets, SMTP settings, Postgres settings, Sentry, and Docker image names.
- Several default secret values use `changethis` and must be replaced for deployment.
