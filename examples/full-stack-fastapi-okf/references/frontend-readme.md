# Frontend README

## Relevant Facts

- The frontend uses Vite, React, TypeScript, TanStack Query, TanStack Router, and Tailwind CSS.
- Local frontend development uses `bun install` and `bun run dev`.
- The generated client lives in `frontend/src/client`.
- Backend OpenAPI schema changes require regenerating and committing the frontend client.
- Playwright tests require the Docker Compose backend stack.
