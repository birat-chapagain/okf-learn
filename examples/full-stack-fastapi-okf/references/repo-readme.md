# Repository README

## Relevant Facts

- The template uses FastAPI, SQLModel, Pydantic, PostgreSQL, React, Vite, Docker Compose, Pytest, Playwright, Traefik, and GitHub Actions.
- The repository includes backend, frontend, Docker Compose files, deployment docs, development docs, and package metadata.
- Some deployment secrets and default values must be changed before deployment.
