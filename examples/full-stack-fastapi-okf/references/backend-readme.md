# Backend README

## Relevant Facts

- Backend dependencies are managed with `uv`.
- Models live in `backend/app/models.py`.
- API endpoints live in `backend/app/api/`.
- CRUD utilities live in `backend/app/crud.py`.
- Backend tests run with `bash ./scripts/test.sh`.
- Running stack tests can use `docker compose exec backend bash scripts/tests-start.sh`.
